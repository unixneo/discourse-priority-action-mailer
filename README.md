# discourse-priority-action-mailer

```

  DISCOURSE_SMTP_PORT_PRIORITY: 
  DISCOURSE_SMTP_USER_NAME_PRIORITY: 
  DISCOURSE_SMTP_PASSWORD_PRIORITY: 
  #DISCOURSE_SMTP_ENABLE_START_TLS_PRIORITY: true        # (optional, default true)
```
